To install in nopCommerce:

1. Set aside your settings for the plugin (necessary until there is an in-place upgrade feature).
2. Uninstall current version if there is already one installed from nopCommerce's admin.
3. Copy the Shipping.SeeSharpShipUsps folder to your nopCommerce Plugins folder.
4. Install the new version from nopCommerce's admin.
5. Reconfigure module.
